BULKMAILER: It is a react-based website which can be used for bulk mailing that means mailing more than one person at a time. Solves the problem of time consumption in sending mails seperately to different people. 

Features:
1. SignIn/SignUp
2. Create groups of contacts
3. Template Based Mails
4. Scheduling of mails
5. Ability to bulk upload data from CSV, Excel
6. Mail History
7. User profile

Tech-Stacks :
Frontend : HTML, CSS, JS, ReactJS, Redux 
Backend  : Django, Celery, Nginx, Gunicorn, PostgreSql 
Design   : Figma
